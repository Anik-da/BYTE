# 🖥️ Frontend

Web-based user interface for BYTE, built with **React + Vite + TypeScript**.

## Structure
- `src/` — Components, pages, hooks, and styles
- `public/` — Static assets served directly

## Setup
```bash
npm install
npm run dev
```

## Tech
- React 18+
- Vite
- TypeScript
- CSS Modules / Vanilla CSS
