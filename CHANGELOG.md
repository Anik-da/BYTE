# Changelog

All notable changes to BYTE will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-26

### Added
- **Desktop Foundation** — Tauri v2 app with React + TypeScript + Vite
- **Custom Window Management** — Frameless window with custom title bar, minimize/maximize/close controls
- **Theme System** — Dark/Light mode with 6 accent colors (Blue, Purple, Cyan, Green, Orange, Pink)
- **Navigation** — Collapsible sidebar, top bar, status bar, floating dock
- **Pages** — Splash screen, login screen, home dashboard, settings, profile
- **Command Palette** — Ctrl+K triggered command palette with search and keyboard navigation
- **Notification Center** — Slide-in panel with notification types and management
- **Settings** — 8 tabs including fully functional Appearance tab with theme/accent/animation controls
- **Dashboard** — 9 module placeholder cards with status badges and staggered animations
- **UI Components** — Button, Card, Input, Avatar, Badge, Toggle, Tooltip, GlassPanel
- **State Management** — Zustand stores for theme, auth, app state, notifications, commands
- **Design System** — Glassmorphism, Inter font, CSS custom properties, responsive layout
- **Rust Backend** — System info and app version Tauri commands
- **Project Structure** — Feature-first architecture with 10 future module placeholders
